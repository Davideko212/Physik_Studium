Eine [[Menge]] von [[Vektor|Vektoren]] $V := \{\textbf{v}_1, \dots, \textbf{v}_{n}\} \subseteq \mathcal{V}$ heißt ein Erzeugendensystem von einem [[Vektorraum]] $\mathcal{V}$ wenn jeder [[Vektor]] $\textbf{v} \in \mathcal{V}$ als
$$\textbf{v} = \sum_{i=1}^n s_{i} \textbf{v}_{i} \ \ \ \text{mit} \ s_{i} \in \mathbb{K}$$
d.h. als eine [[Linearkombination]] der [[Vektor|Vektoren]] in $V$ dargestellt werden kann. Somit spannt die [[Lineare Hülle]] der [[Vektor|Vektoren]] in $V$ den [[Vektorraum]] $\mathcal{V}$ auf.

Ein "minimales" Erzeugendensystem ist eine [[Basis]].
