Sei $A \in \mathbb{K}^{n \times n}$ eine [[Hermitesche Matrix]]. Es heißen
$$M_{1} = a_{11}, \ \ \ M_{2} = \det
\begin{pmatrix}
a_{11} \ a_{12} \\
a_{21} \ a_{22}
\end{pmatrix},
\ \ \ \dots,
\ \ \ M_{n} = \det(A)
$$
die Hauptminoren der Matrix $A$. Gilt für alle $k = 1, \dots, n$
$$ M_{k} > 0$$so heißt die Matrix positiv [[Definitheit|definit]]. Gilt für alle $k = 1, \dots, n$
$$(-1)^k M_{k} > 0$$
so heißt die Matrix negativ [[Definitheit|definit]].
