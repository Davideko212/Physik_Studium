Seien $\textbf{v}_1, ..., \textbf{v}_k \in \mathcal{V}$. Die Menge $\mathcal{L}(\textbf{v}_1, ..., \textbf{v}_k)$ aller [[Linearkombination|Linearkombinationen]] von $\textbf{v}_1, ..., \textbf{v}_k$ heißt die lineare Hülle von $\textbf{v}_1, ..., \textbf{v}_k$.

Der linearen Hülle kommt eine weitere Bedeutung zu, denn sie ist keine beliebige [[Teilmenge]] eines [[Vektorraum|Vektorraums]]:
Seien $\textbf{v}_1, ..., \textbf{v}_k \in \mathcal{V}$. Die lineare Hülle $\mathcal{L}(\textbf{v}_1, ..., \textbf{v}_k)$ ist ein [[Unterraum]] von $\mathcal{V}$.
